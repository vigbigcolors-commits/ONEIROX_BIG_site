Privacy Policy

Oneirox is built without accounts. We collect the minimum data needed to run the decoder and improve the service.

What You Submit

When you use Decode, your dream text is sent to our API to generate a reading. We do not require registration, email, or payment to use the core decoder.

Local Storage

The Sensory Dream Mapper may save somatic context in your browser's localStorage (key: onx_mapper_data) for up to 30 minutes so Decode can include body-signal context. This data stays on your device until you clear it or it expires.

Server Logs

Our hosting and API providers may log standard request metadata (IP address, timestamp, user agent) for security and reliability. We do not sell personal data.

Third Parties

Dream analysis is processed via our API infrastructure. Share features (email, social) use third-party services only when you explicitly choose to share a reading.

Contact

Questions about privacy: use the contact method listed on About when available.
